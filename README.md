This unsupervised machine learning project focuses on analyzing the Mice Protei Expression dataset to identify hidden patterns and proteing segments using clustering techniques. The project involved data preprocessing, exploratory data analysis, feature scaling, and the implementation of unsupervised learning models to better understand mice protein expression data.

Clustering algorithms such as K-Means and Hierarchical Clustering were applied to group customers based on similarities in purchasing patterns and spending behavior. Data visualization and statistical analysis techniques were used to interpret cluster characteristics and evaluate the effectiveness of the segmentation process.

The project demonstrates practical applications of unsupervised learning in mice protein analytics and segemntation while strengthening skills in Python, data analysis, clustering techniques, and data-driven decision-making.
